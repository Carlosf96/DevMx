# DEV MX

A social platform created by devs for devs to connect devs.

## Installation

clone repo

cd into repo dir

inside of repo do the following commands

```bash
$ npm install && npm run client-install
$ npm run dev
```
## Built With

* [MongoDB]
* [ExpressJS]
* [ReactJS]
* [Redux]
* [NodeJS]
* [Boostrap]
